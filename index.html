Opening departmental stores in optimised places of Bangalore

IBM Data Science Capstone Project

In this project, I had to extract names of different neighborhoods of Bangalore from Wikipedia using BeautifulSoup library and generate their latitudes and longitudes, explore their venues using FourSquare API and analyze which areas have less amount of departmental store. Using Folium library I generated the map of Bangalore and used various bar charts to visualize the data. Using Kmeans clustering, I clustered neighborhoods depending on the similarity in their venues, and predicted the best neighborhood for opening a departmental store. A complete documentation with code is present in my GitHub profile.

Face Recognition using SVM PCA

OpenCV Project

In this project, I used OpenCV library for image pre-processing, detecting and recognizing human faces. I used the Haar Cascade Frontal Face Classifier to detect the faces, used Principal Component Analysis for dimensional reduction of images and finally used Support Vector Classifier with RBF Kernel to classfy different faces. A complete documentation with codes is present in my GitHub profile.
